Linux Commands

  1. ls - list file names in present directory
  2. ls -al - list file names including all informations about permissions and types
  3. cd - change directory (go to home)
  4. cd .. - change directory back to one step before
  5. cd / - go to the root directory
  6. pwd - present working directory
  7. whoami - check the current user
  8. reboot - restart the system
  9. poweroff - shutdown the system
  10. sudo - as root
  11. sudo -i - change user as root
  12. mv - move file
  13. cp - copy file
  14. rm - remove file
  15. rm -rf - remove the directory
  16. mkdir - make or create directory
  17. mv filename nameTochange - change the file name
  18. clear - clear the terminal
  19. exit - exit the terminal
  20. history - show all commands used
  21. chmod - change mode (change permission)
      example
    • sudo chmod +x filename - permission as read and execute
    • sudo chmod 777 filename - full permission read,write and execute
      • 3 types of permission group
      • Owner - who created file
      • User - who is loggined
      • Other - who is currently not loggined
  22. code . - open vscode in folder in which the project files exist
  23. sudo pacman -S softwareName - install package(software) from pacman
  24. sudo snap install softwareName --classic - install software from snapcraft

Linux shortcut keys

  1. ctrl + alt + t - open terminal
  2. ctrl + l - clear terminal
  3. ctrl + c - stop running command in terminal
  4. ctrl + d - exit the terminal
  5. ctrl + u - clear the line in terminal
  6. ctrl + a - go to the start of line in terminal
  7. alt + f4 - close the window(opened program)
  8. alt + tab - swap the programs opened
  9. ctrl + s - save file
  10. ctrl + a - select all files or text in text editor
  11. ctrl + c - copy the selected file
  12. ctrl + v - paste the copied file to the current directory
  13. ctrl + x - move the file
  14. press up and down arrows in terminal to get commands that have been used
